# Training vs Fine-tuning
- SGD(경사하강법)등 Optimizer알고리즘을 통해 Parameter를 바꾸는것은 동일
- Training은 처음부터 학습을 시켜서 최적의 Parameter를 구하는 과정
- Fine-tuning은 이미 학습된 모델을 가져와서 우리의 제품에 맞도록 학습을 다시 시켜서 Parameter를 수정하는것
- Fine-tuning도 동일하게 Optimizer알고리즘을 사용하지만, 출발점이 이미 학습된 Parameter이며, Learning Rate를 더 작게 설정. 일부 Parameter만 학습시키는 경우도 있음.

# 경사하강법 vs loss Function
- 둘은 별도의 개념이 아니라 loss Function을 미분해서 기울기가 0이 되는 지점을 찾는것이 경사하강법
- 새 Parameter = 현재 Parameter - Learning Rate * 기울기
