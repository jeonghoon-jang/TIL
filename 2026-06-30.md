# GD(Gradient Descent) VS SGD(Stochastic Gradient Descent) 
- 계산 효율/속도, 메모리 절약,local minimum 탈출의 이유로 SGD 사용
# SOTA(State Of The Art)모델 
- 현재 특정 Task에서 가장 성능이 좋은 모델/방법
# Ranking, Retrieval
- Retrieval(검색) 많은 문서들 중 질문과 관련된것을 찾아오는 Task
- Ranking(순위) 찾아온것들을 관련도 순으로 줄세우는 Task
# 모델의 평가지표
- metric
- metric의 핵심조건: 실제 성능 반영
- METEOR, BLEU, ROUGE 등 Metric -> 어떤 데이터로 어떻게 측정하냐에 따라 결과가 달라질 수 있음
- NLP 벤치마크 -> 동일한 조건에서 평가되기 때문에, 객관적으로 성능을 비교할 수 있음
-  GLUE (General Language Understanding Evaluation) -> NLU(자연어이해) 평가를 위한 벤치마크
