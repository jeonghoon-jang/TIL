# Attention VS Transformer VS GPT
- Attention: 출력의 각 step에서 입력 중 어디에 집중할지 가중치로 골라 관련 정보를 가져오는 매커니즘
- Transformer: Attetion을 기반으로 활용한 모델
- GPT: Transformer기반 모델

# Seq2Seq
- 시퀀스를 받아 시퀀스를 생성하는 시퀀스 변환 모델
- 인코더(입력 이해) + 디코더(출력 생성)로 구성
- 입력 이해 / 출력 생성으로 역할 분리 → 학습·설계에 용이
- 입출력 길이가 다른 것에 대처 가능

# Seq2Seq VS Attention
- 초기 Seq2seqs: 인코더 입력 전체를 고정 크기 context 하나로 압축
- → 문장이 길수록 정보 병목, 앞쪽 정보가 뒤로 갈수록 희미해짐
- 해결 1: 매 step 정보를 주자
- 문제: step마다 중요한 정보(context)가 다름
- 해결 2(Attention): context를 그냥 전달하지 않고,
  이전 문맥에서 '관련 있는 부분에 가중치'를 부여해 
  step마다 다른 context vector를 만들어 전달
